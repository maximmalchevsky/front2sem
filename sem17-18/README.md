# Требования:
- Docker (версия 20.10.0 или новее)
- Docker Compose (версия 1.29.0 или новее)
- 4 ГБ ОЗУ
- 2 CPU 
- 500 МБ свободного места на диске

# Запуск программы:
```bash
git clone https://github.com/maximmalchevsky/front2sem
```
```bash
cd front2sem
```
```bash
cd sem17-18
```
```bash
docker compose up --build -d
```
